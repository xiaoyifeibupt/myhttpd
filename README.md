# xhttpd

`xhttpd`是一个简单高效的http server，C++语言开发，为了练习所学的一些知识：http 协议、网络编程、I/O复用，多线程以及C++ 11新特性。

-	2015.5.5 出差一周，进行了非阻塞IO的学习，epoll的使用
-	2015.5.4 完成socket接口的封装
-	2015.5.1 准备活动，关于文件读取，缓冲区，log设计和实现




##功能

-	支持GET
-	支持文件和目录的访问
-	非阻塞I/O
-	epoll
-	支持403,404,500
